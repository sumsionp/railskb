== README

This is a Rails based knowledge base application designed to be very simplistic
to enter information. This makes it so the program doesn't get in the
way when the user wants to use it.

Built with Ruby 2.1.1

=== Deployment

After cloning, run the following commands:

+bundle+
+rake db:migrate+
+rails server+

=== Testing

RSpec is used for testing. To run all of the tests, run the following command
from the root of the project:

+rspec+
